# 📚 LMS Técnicas - Sistema de Aprendizaje Interactivo

Sistema de gestión de aprendizaje (LMS) con 100 técnicas de lenguaje, morfología, gramática y estándares educativos de Colombia.

## 🎯 Características

- **100 técnicas educativas** en lenguaje, morfología, gramática y estándares MEN
- **Dashboard para estudiantes** con seguimiento de progreso
- **Panel para profesores** con estadísticas de avance
- **Ejercicios interactivos** para cada técnica
- **Sistema de autenticación** con roles (estudiante/profesor)
- **Base de datos SQLite** ligera y portátil

## 🛠️ Tecnologías

- Django 6.0
- SQLite
- HTML5 + CSS
- BeautifulSoup4 (para carga de datos)
- Python 3.12+
